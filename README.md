
# Mental Health Treatment Access Analysis - Argentina

## Descripción del proyecto

Este proyecto tiene como objetivo analizar el acceso a tratamientos de salud en personas con diagnosticos relacionados a la salud mental en Buenos Aires, Argentina.

El análisis busca identificar barreras en el acceso a tratamientos, el rol de las obras sociales y prepagas, y la intervención del Estado.

## Fuente de datos

Los datos fueron recolectados mediante un formulario diseñado específicamente para este estudio y no incluyen datos personales que puedan identificar a los participantes.  

El formulario incluye información sobre:

- Lugar de residencia
- Diagnóstico confirmado
- Tipo de discapacidad
- Necesidad de tratamientos
- Certificado Único de Discapacidad (CUD)
- Cobertura médica
- Acceso a tratamientos
- Recursos de amparo
- Cobertura estatal

## Metodología de recolección de datos

Los datos fueron recolectados mediante un formulario distribuido a través de redes sociales y grupos de WhatsApp relacionados con la temática del proyecto, así como en centros y espacios vinculados a la atención de la salud mental y la discapacidad.

## Herramientas utilizadas

- SQL
- Power BI
- Excel / Google Sheets
- GitHub

## Objetivos del análisis

- Analizar el acceso a tratamientos
- Evaluar el rol de las coberturas médicas
- Identificar barreras en el acceso a la salud
- Analizar el impacto del Certificado Único de Discapacidad
- Evaluar el rol del Estado
